Uses prettier, rollup & esbuild for the example. Nice defaults. Good stuff.

## Install

```sh
yarn add @charliewilco/module
```

## Usage

```ts
import { SomeClass } from "@charliewilco/module";

const c = new SomeClass();

c.useMethod();
```
